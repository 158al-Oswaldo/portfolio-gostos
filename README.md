# 🌐 Site Estático com HTML e CSS

Este projeto consiste em um site estático desenvolvido utilizando **HTML** e **CSS**, com foco na organização de layout através de **DIVs** e **Flexbox**.

O objetivo do projeto é praticar conceitos fundamentais de estruturação de páginas web e estilização responsiva.

---

## 🚀 Funcionalidades

- Estruturação de páginas com HTML
- Estilização utilizando CSS
- Organização de layout com `div`
- Alinhamento e posicionamento com `Flexbox`
- Interface simples e responsiva

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3

---

## 📚 Conceitos Praticados

Durante o desenvolvimento deste projeto foram aplicados conceitos como:

- Estrutura semântica em HTML
- Seletores e propriedades CSS
- Flexbox para alinhamento de elementos
- Espaçamento e posicionamento
- Responsividade básica
- Organização visual de componentes

---


## 💻 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de aprimorar habilidades em desenvolvimento front-end, especialmente na criação e organização de layouts utilizando Flexbox.
